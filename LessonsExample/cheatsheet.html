Document Structure

<html></html>

Root element of the HTML page

<head></head>

Contains meta info, title, links — not displayed

<title></title>

Page title shown in browser tab & bookmarks

<body></body>

All visible page content goes here

Tt

Text Tags

<h1> – <h6>

Headings — H1 largest, H6 smallest

<p></p>

Creates a paragraph

<strong> / <em>

Bold / Italic emphasis (semantic)

<pre> / <code>

Preformatted or monospace code text

<br>

Line break (no closing tag)

<blockquote>

Indented block quote from both sides

🔗

Links

<a href="URL">

Creates a hyperlink to a URL

<a href="mailto:...">

Hyperlink that opens an email client

<a name="NAME">

Creates an anchor/target within the page

<a href="#NAME">

Links to an anchor on the same page

🖼

Images

<img src="URL" />

Embeds an image from a URL

alt=?

Alternate text — required for accessibility (ADA)

width=? height=?

Sets image dimensions in pixels

<hr>

Horizontal rule divider line

☰

Lists

<ul></ul>

Unordered (bulleted) list

<ol start=?></ol>

Ordered (numbered) list — start sets first number

<li></li>

Individual list item

<dl> <dt> <dd>

Definition list with terms and descriptions

⬜

Layout & Formatting

<div></div>

Block-level container, styled with CSS

<span></span>

Inline container, styled with CSS

⊞

Tables (for data)

<table>

Creates a table

<tr>

Table row

<td>

Table cell

<th>

Header cell — bold & centered

Attributes: border, cellspacing, cellpadding, colspan, rowspan, width

📝

Forms

<form></form>

Defines a form

<input type="text">

Single-line text field

<input type="checkbox">

Checkbox — add checked to pre-check

<input type="radio">

Radio button

<input type="submit">

Submit button — value sets label

<textarea>

Multi-line text box — cols × rows

<select> <option>

Dropdown or scrolling menu

HTML5 input types:

emailurlnumber rangedatesearchcolor

Font Properties

font-family

Sets typeface: font-family: Arial, sans-serif;

font-size

Size in px, pt, em, %, or keywords like large

font-weight

normal, bold, or numeric 100–900

font-style

normal, italic, oblique

font-variant

normal or small-caps

font

Shorthand: italic bold 12pt/14pt Times, serif

🎨

Color & Background

color

Text color — name or hex: color: #000080;

background-color

Element background color

background-image

url(/path/to/img.gif)

background-repeat

repeat, repeat-x, repeat-y, no-repeat

background-attachment

scroll or fixed

background

Shorthand combining all background properties

¶

Text Properties

text-align

left, center, right, justify

text-decoration

underline, overline, line-through, none

text-transform

uppercase, lowercase, capitalize

text-indent

First-line indent: text-indent: 5em;

line-height

Line spacing: line-height: 200%;

letter-spacing

Space between chars: 0.1em

word-spacing

Space between words: 0.4em

vertical-align

middle, top, super, or %

⊡

Box Model

margin

Outside spacing — shorthand: top right bottom left

padding

Inner spacing between content and border

width / height

px, %, or auto

float

left, right, none — wraps text around element

clear

Prevents floating elements on sides: left, right, none

▣

Borders

border

Shorthand: 10px solid red

border-width

Thickness — one value or top/right/bottom/left

border-style

solid, dashed, dotted, double, none

border-color

Color by name or hex value

Individual sides: border-top, border-right, border-bottom, border-left
