Module 1 Portfolio Challenge: About Me Section

The Web & HTML Basics
📌 What you're building Create the very first page of your portfolio with your name, a big welcome heading, and a short paragraph about yourself.
Steps
  • 1Create a new file called index.html and add the basic HTML structure: <!DOCTYPE html>, <html>, <head>, and <body>.
  • 2Inside <head>, add a <title> tag with your name — this shows up in the browser tab.
  • 3Inside <body>, add an <h1> with your name, an <h2> that says something like "Welcome to my portfolio!", and a <p> tag with 2–3 sentences about who you are and what you love.
  • 4Add a second <p> below with your grade and one fun fact about yourself.
Starter Code
<!-- index.html — start here! -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title><!-- ADD YOUR NAME HERE --></title>
</head>
<body>

  <!-- ADD YOUR CODE HERE -->

</body>
</html>
Expected Result
Your browser shows a plain white page with your name as the biggest heading at the top, a welcome message below it, and two paragraphs of text about you. The browser tab shows your name.
⭐ Bonus
Extra challenge: Add a third heading <h3> with the label "My Favourite Things" and use three <p> tags to list your top three hobbies, foods, or movies.

Module 2 Portfolio Challenge: Links & Styled Text

Links, Colors & Fonts
📌 What you're building Add a "Find Me Online" section with clickable links to your favourite websites, plus make your page pop with colour and custom fonts using CSS.
Steps
  • 1Below your About Me section, add an <h2> that says "Find Me Online".
  • 2Add at least three <a href="..."> links — these could be YouTube, a favourite game site, or a school project. Make each link open in a new tab using target="_blank".
  • 3Inside <head>, add a <style> tag. Use CSS to change the body background colour (try a hex code like #f0f4ff) and set a font family like font-family: Arial, sans-serif;.
  • 4Change the colour of your h1 heading using the CSS color property with a hex code you like. Try #2d6be4 for a nice blue!
  • 5Add a special character somewhere on your page — try a copyright symbol &copy; at the very bottom, or a heart &hearts; next to your name.
Starter Code (includes Module 1 output)
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Alex's Portfolio</title>
  <!-- ADD YOUR <style> TAG HERE -->

  <h1>Alex Johnson</h1>
  <h2>Welcome to my portfolio!</h2>
  <p>Hi! I'm Alex, a 7th grader who loves coding, drawing, and video games.</p>
  <p>I'm in 7th grade and my fun fact is that I can solve a Rubik's cube in under 2 minutes!</p>

  <!-- ADD YOUR "Find Me Online" SECTION HERE -->

</body>
</html>
Expected Result
Your page now has a colourful background, a styled heading in your chosen colour, and a "Find Me Online" section with clickable links. Clicking a link opens that website in a new browser tab.
⭐ Bonus
Extra challenge: Use CSS to make the link colour something other than the default blue — try a { color: #e91e8c; } — and also remove the underline from links using text-decoration: none;.

Module 3 Portfolio Challenge: Skills Section

Introduction to CSS
📌 What you're building Add a "My Skills" section and move all your CSS into its own separate stylesheet file, keeping your code neat and organised.
Steps
  • 1Create a new file called style.css in the same folder as your index.html. Move all your CSS from the <style> tag into this file, then delete the <style> tag and replace it with <link rel="stylesheet" href="style.css">.
  • 2In index.html, add a new section using a <div> tag with class="skills-section". Inside it, put an <h2> saying "My Skills" and use <span> tags to label each skill (e.g. <span class="skill">HTML</span>).
  • 3In style.css, use the class selector .skills-section to add a background colour and some padding to the section.
  • 4Style the .skill class to give each skill badge a different background colour, rounded corners (border-radius), and some padding so they look like little chips.
  • 5Give your page an id="main-heading" on the <h1> and style it by ID in CSS using #main-heading { }.
Starter Code (includes Modules 1–2 output)
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Alex's Portfolio</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <h1 id="main-heading">Alex Johnson</h1>
  <h2>Welcome to my portfolio!</h2>
  <p>Hi! I'm Alex, a 7th grader who loves coding, drawing, and video games.</p>
  <p>I'm in 7th grade and my fun fact is that I can solve a Rubik's cube in under 2 minutes!</p>

  <h2>Find Me Online</h2>
  <a href="https://youtube.com" target="_blank">YouTube</a>
  <a href="https://scratch.mit.edu" target="_blank">Scratch</a>

  <!-- ADD YOUR SKILLS SECTION HERE -->

</body>
</html>
Expected Result
Your page now has a "My Skills" section with colourful badge-style labels for each skill. Your CSS lives in its own file and the page still looks the same (or better!). Each skill chip has its own background colour and padding.
⭐ Bonus
Extra challenge: Wrap all the content above "My Skills" in a <div class="intro"> and give it a different background colour than the skills section, creating a two-tone page layout.

Module 4 Portfolio Challenge: Photo Gallery

Images & Media
📌 What you're building Add a "My Gallery" section that shows off images of things you love — hobbies, places, or your favourite things — with captions.
Steps
  • 1Add a new <div class="gallery"> section after your Skills section with an <h2> heading "My Gallery".
  • 2Add at least three images using <img src="..." alt="...">. You can use free image URLs from the web (right-click an image → "Copy image address") or save images into your project folder. Always write a meaningful alt description!
  • 3Below each image, add a short <p> caption describing what it shows.
  • 4In style.css, set a max-width on your images so they don't stretch the whole page (try max-width: 300px;). Also add a CSS background image to your page header area using background-image: url(...);.
Starter Code (includes Modules 1–3 output)
<!-- index.html — add the gallery section below skills -->
  ...
  <div class="skills-section">
    <h2>My Skills</h2>
    <span class="skill">HTML</span>
    <span class="skill">CSS</span>
    <span class="skill">Drawing</span>
  </div>

  <!-- ADD YOUR GALLERY SECTION HERE -->
  ...
Expected Result
Your portfolio now has a gallery of images, each sized nicely at around 300px wide with a short caption underneath. Your page header area has a background image. All images have helpful alt text for screen readers.
⭐ Bonus
Extra challenge: Wrap each image and its caption inside a <div class="gallery-item"> and in CSS give each item a border and a slight box-shadow to make them look like photo prints.

Module 6 Portfolio Challenge: Styled Layout

Advanced CSS
📌 What you're building Transform your portfolio from a plain list of sections into a properly laid-out page with a sticky header, a sidebar, and interactive hover effects.
Steps
  • 1Wrap your entire page content in a <div class="page-wrapper">. Then create a <header> at the top with your name and navigation links. In CSS, use position: sticky; top: 0; so it sticks to the top as visitors scroll.
  • 2Create a two-column layout by adding a <div class="main-content"> and a <div class="sidebar">. Use float: left on the main content and float: right on the sidebar. Set width on each (e.g. 65% and 30%). Move your skills badges and links into the sidebar.
  • 3Add a :hover pseudo-class to your skill badges — change their background colour when someone hovers over them.
  • 4Style your table rows with alternating colours using the tr:nth-child(even) selector to make the table easier to read.
  • 5Use the CSS box model to add proper padding and margin around each major section so the page doesn't feel cramped.
Starter Code (includes Modules 1–5 output)
<!-- Restructure your existing page like this: -->
<body>

  <header class="site-header">
    <!-- ADD YOUR NAME + NAV LINKS HERE -->
  </header>

  <div class="page-wrapper">

    <div class="main-content">
      <!-- About Me, Gallery, Table, Audio go here -->
    </div>

    <div class="sidebar">
      <!-- Skills + Links go here -->
    </div>

  </div>

</body>
Expected Result
Your portfolio now looks like a real website! There's a sticky navigation bar at the top, a two-column layout with main content on the left and a sidebar on the right, zebra-striped table rows, and skill badges that change colour when you hover over them.
⭐ Bonus
Extra challenge: Add a z-index to your sticky header (try z-index: 100) and give it a semi-transparent background so content "slides under" it as you scroll — use background: rgba(255,255,255,0.9); and add backdrop-filter: blur(4px); for a frosted-glass look.

HTML Forms

Steps
  • 1At the bottom of your main-content div, add an <h2> "Contact Me" and a <form> tag. Set method="post" on the form (even though we're not connecting it to a server yet — great practice!).
  • 2Add a text input for the visitor's name: <input type="text" name="name" placeholder="Your name">. Include a <label> above it.
  • 3Add an email input: <input type="email" name="email" placeholder="your@email.com">. The browser will automatically check it looks like an email address!
  • 4Add a subject dropdown using <select> with at least three <option> choices — like "Just saying hi!", "Question about your portfolio", and "Collaboration idea".
  • 5Add a <textarea name="message" rows="5"> for a longer message, then a submit button: <input type="submit" value="Send Message">. Style the form in CSS to make it look neat — add padding, a border, and a background colour to the form container.
